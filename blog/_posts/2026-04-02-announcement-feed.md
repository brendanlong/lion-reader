This feed will contain Lion Reader service announcements and occasional feature updates. Feel free to unsubscribe if you have no interest in this.

Some features you should be aware of if you haven't seen the demo site:

- [Email newsletters](https://lionreader.com/demo/all?entry=email-newsletters)
- [AI summaries](https://lionreader.com/demo/all?entry=ai-summaries) (if you add an API key)
- [Save for later](https://lionreader.com/demo/all?entry=save-for-later) + [browser extensions](https://lionreader.com/demo/all?entry=browser-extension) (and [a Discord bot](https://lionreader.com/demo/all?entry=discord-bot))
- APIs for native apps ([Google Reader / FreshRSS](https://lionreader.com/demo/all?entry=google-reader-api) + [Wallabag](https://lionreader.com/demo/all?entry=wallabag-api))
- [MCP server](https://lionreader.com/demo/all?entry=mcp-server) for AI usage (ask your AI to upload its reports to Lion Reader!)

This feed will only contain major announcements. For more frequent, detailed updates, see the [GitHub releases feed](https://github.com/brendanlong/lion-reader/releases.atom).

Also, if you ever have any bug reports or feature requests, please [report them on GitHub](https://github.com/brendanlong/lion-reader/issues) or [email me](mailto:self@brendanlong.com).
